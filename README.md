# 党建信息平台 (Party Building Platform)

一个面向基层党组织的党员信息管理平台，支持党员档案管理、培养历程追踪、思政日历和数据分析画像。

## 功能模块

### 首页仪表盘
- 党员总数及各阶段人数统计
- 发展阶段分布环形图
- 近期日程预览
- 党员动态更新列表

### 党员管理
- 党员信息列表（姓名、性别、部门、阶段、入党时间等）
- 按姓名/部门搜索
- 点击查看党员完整档案

### 培养历程
- 按党员切换查看个人发展时间线
- 直观展示：入党申请人 → 积极分子 → 发展对象 → 预备党员 → 正式党员
- 每个阶段列出所提交的材料，点击可查看材料原文内容

### 思政日历
- 全面日历视图，支持前后翻月和跳转当月
- 自动标注五大类型日程：
  - 党的纪念日（建党节、建军节、青年节等）
  - 党的重大会议（历次中央全会、全国两会、全国组织工作会议等）
  - 法定节假日
  - 传统节日
  - 自定义日程（用户可手动添加）

### 数字画像
- 统计概览卡片（党员总数、已转正、在途培养、平均材料数、志愿总时长等）
- 发展阶段分布环形图（SVG 渲染）
- 各阶段人数对比柱状图
- 志愿活动记录管理（支持添加和删除）

## 使用方法

直接在浏览器中打开 `index.html` 即可使用（纯前端应用，无需服务器）。

如需部署到 GitHub Pages：

```bash
# 将项目推送到 GitHub 仓库后，在仓库设置中：
# Settings > Pages > Source: 选择 main 分支，根目录
git add .
git commit -m "init: 党建信息平台"
git remote add origin https://github.com/txrainyun/dangjian.git
git push -u origin main
```

访问地址：`https://txrainyun.github.io/dangjian/`

## 技术栈

- 纯前端 HTML + CSS + JavaScript（无需构建工具）
- SVG 图形渲染（环形图、柱状图）
- 数据暂存于 `js/data.js`（模拟数据）
